Picture Shuffle is when the  user clicks within a 5x5 grid trying to unshuffle an image that is broken up into 25 pieces.
There is one piece missing from the picture which is the empty space the pieces can go to.
Pieces may only move horizontally or vertically and only a piece that is directly next to the open space may move there (no jumping).
The user has completed the puzzle when they have correctly arranged the pieces in order and displayed the image.
